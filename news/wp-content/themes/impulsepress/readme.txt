== Changelog ==
1.4.1 - October 25, 2014
* Upgrade to Font Awesome 4.2.0


1.4 - June 14, 2014
* Bug fixes related to the TinyMCE for WP 3.9 - The shortcodes icons are not backwards compatible! Only work from WP version 3.9
* Added shortcode for bootstrap carousel
* Removal of options for colours and font sizes - many users got confused with this
* Introduction of CSS themes

1.3 - March 15, 2014
* Many bug fixes in the options panel
* Added all bootstrap buttons options panel page
* Introduced Custom Meta Boxes
* Enhanced Home Parallax page
* Breadcrumbs are now optional per page
* Possibility to disable styling options and use CSS instead


1.2.5 - March 3, 2014 =
* Many bug fixes
* New carousel gallery type
* Upgraded to Bootstrap 3.1.1
* Multiple galleries in the same page
* Shortcodes enabled inside widgets

1.2 -  January 30, 2014
* Bug fixes
* Masonry Gallery with touch-based lightbox
* Parallax Home Page template
* 1.1 -  January 16, 2014

First version release
* 1.0 - October 9, 2013
